Installing .dll Mods:

Step 1: Locate your Managed directory.
Your Managed directory can be located easily from your Steam install directory. From the Steam directory, go to ‘steamapps’ -> ‘common’ -> ‘Rain World’ -> ‘RainWorld_Data’ -> ‘Managed’.

Step 2: Copy the file ‘Assembly-CSharp.dll’ from your Managed directory to a safe location.
This is your original game file that you will use to restore the game to its original state if you ever need to.

Step 3: Replace ‘Assembly-CSharp.dll’ with your modded .dll. Be sure to rename your .dll to ‘Assembly-CSharp.dll’ so that the game will recognize it.

That’s it, the mod is now installed!

To uninstall the mod and revert the game to its normal state, replace the modded file with your original copy.

Note: Using .dll mods will not modify your saves unless you load them up, so creating a new save especially for a certain mod is recommended. Mod authors cannot guarantee your save’s safety if you load up any save that was created with a different .dll mod in place or the vanilla .dll.

Installing Dev Tools:

See ‘.dll Mods’ above.

Region Revamp Mods:

These mods will usually come with the dev tools dll (necessary in order to prevent the game detecting that the world files were tampered with), a save file that has the player in the pertinent region, and the folder containing the region files.

For these, just install the .dll mod as usual, backup your save (which is located in steamapps\common\Rain World\UserData), and replace it with the new save.

Then go to \steamapps\common\Rain World\World\Regions, backup the appropriate region folder (the one with the same name as the modded one), and replace it with the modded one.
